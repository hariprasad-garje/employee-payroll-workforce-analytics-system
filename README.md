# employee-payroll-workforce-analytics-system
A Python-based Employee Payroll &amp; Workforce Analytics System. It includes employee management, payroll processing, salary analytics, department-wise analysis, employee search, filtering, and file handling.

## Requirements
- Python 3.x
- NumPy

  
Modules:
 • Add employee details (ID, Name, Department, Salary).
 • Store and retrieve records using text files.
 • Create Employee class with OOP concepts.
 • Validate salary and inputs using exception handling.
 • Calculate salary statistics (mean, median, standard deviation, highest, lowest).
 • Apply increments using scalar operations and broadcasting.
 • Perform department-wise matrix operations.
 • Generate random employee datasets for testing.
 • Filter employees based on salary using Boolean indexing.
